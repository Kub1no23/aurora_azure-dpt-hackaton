Demo stránky s FAQ Botem

Tady je klasická stránka — pod ní vložen widget.

Příklad: vložení widgetu do jiné HTML stránky

<head>
  <script src="https://tvoje-domena.cz/static/chatbot.js" defer></script>
</head>
<body>
  ...
  <div id="faq-bot-root"></div>
</body>

Pokud #faq-bot-root není přítomen, widget se automaticky ukotví do pravého dolního rohu.
